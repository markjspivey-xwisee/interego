Interego — three apps, one pod

Same pod content. Three different apps presenting it through three different lenses. Zero app-specific schema.

Pod content

App 1. Reads the same descriptors, presents them as a timeline. Asks: "what did I do recently?"
App 2. Same descriptors, grouped by subject; surfaces who attested what about whom. Asks: "who's vouching for what?"
App 3. Same descriptors, rendered as raw RDF triples. Asks: "show me the underlying graph."
The point: none of these apps wrote anything app-specific to the pod. None required prior coordination with the pod's author. None know which app produced which descriptor. They all read the same standard cg: + W3C vocabulary shape — Verborgh's interop holy grail.   ·   github
