Pod Federation

Decentralized pods discovered across the federation. Each pod is an autonomous Solid container with its own context descriptors, agent registries, and access control.

Context Descriptors

All descriptors discovered across the federation. Each descriptor annotates a Named Graph with typed facets: temporal validity, provenance, semiotic frame, trust level, access control, and federation metadata.

Trust Chain

      
Trust levels across the federation. Each descriptor declares its trust basis: SelfAsserted (unverified claim), ThirdPartyAttested (external attestation), or CryptographicallyVerified (signed proof).

    

    

  


  
  

    

      
Algebraic Composition

      
Compose context descriptors from two pods using algebraic operators. Union (A + B), Intersection (A * B), Restriction (A | B), Override (A >> B). The operators form a bounded lattice with well-defined merge semantics per facet type.

SPARQL Query

      
Execute SPARQL queries against the materialized PGSL triple store. The triple store contains all atoms, fragments, levels, constituents, and provenance from the lattice.

    

    
    

      
      
    

    

  


  
  

    

      
SHACL Validation

      
Three-layer SHACL validation of the PGSL lattice: Core (node constraints), Structural (lattice invariants), Domain (custom shapes).

      
    

    

  


  
  
  

    

      
Coherence Coverage

      
Usage-based coherence between agents. Meaning is usage — two agents are coherent when they use shared signs in the same syntagmatic contexts. The dangerous state is unexamined: agents proceeding as if aligned without verification.

      
    

    

  


  

    

      
Decision Functor (OODA Loop)

      
Observe → Orient → Decide → Act. The decision functor maps observation presheaves to action categories. Each agent extracts observations from its lattice, orients via coherence verification, selects a strategy (exploit/explore/delegate/abstain), and produces ranked affordances.

      
    

    

  


  

    

      
PROV Action Traces

      
W3C PROV-O audit trail of every action across the federation. Every ingest, compose, constrain, and discover is recorded with agent, timestamp, and policy decision.
