lpc: — Learner / Performer Companion (vertical application) lpc:

Namespace https://markjspivey-xwisee.github.io/interego/applications/learner-performer-companion/lpc# · 30 terms.

Vertical-scoped, non-normative vocabulary for human learners / workers managing their own portable wallet of credentials, learning history, training content, and performance records. NOT part of the Interego protocol.

Machine-readable form: lpc.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. lpc:answeredQuestion = https://markjspivey-xwisee.github.io/interego/applications/learner-performer-companion/lpc#answeredQuestion).

Terms

lpc:answeredQuestion Datatype property
The user question this cited response is answering. Stored verbatim so the response stays interpretable when re-read later.
lpc:assistantAttribution Object property
DID of the Interego-grounded assistant that produced this response. Distinct from the user's DID — the response is the assistant's claim about the user's data, not the user's own claim.
lpc:authoritativeSource Object property
Issuer / publisher of the training content. Treated as authoritative for citation purposes.
lpc:basedOnStatement Object property
IRI of the source xAPI Statement (typically a urn:iep:lrs-statement:* form produced by lrs-adapter ingest).
lpc:CitedResponse Class
An assistant chat response that cites grounding fragments (training content, credentials, performance records). Stored as a descriptor so the citation chain is auditable. The user can click through any citation to the source descriptor.
lpc:citesDescriptor Object property
Cited a whole descriptor (credential, performance record, learning experience) rather than a fragment.
lpc:citesFragment Object property
PGSL atom or descriptor that grounds part of this response. Multiple per response is normal.
lpc:contentFormat Object property
lpc:ContentFormat Class
lpc:contentStandard Datatype property
Higher-level standard the content conforms to (e.g., 'TLA-LAP', 'SCORM-2004-3', 'cmi5-1.0').
lpc:Credential Class
A verifiable credential / Open Badge 3.0 / IMS CLR 2.0 assertion / IEEE LERS recognition record. Carries the underlying VC proof block plus verification status. Issuer attribution lives in iep:TrustFacet.issuer.
lpc:credentialFormat Object property
lpc:CredentialFormat Class
Enumerated credential format. Each named individual identifies one of the standards-side recognition formats this vertical can hold.
lpc:credentialFramework Datatype property
Version string of the underlying recognition format (e.g., 'OB-3.0', 'VC-2.0', 'CLR-2.0', 'LERS-WG-draft-2026').
lpc:DevelopmentPlan Class
User-side plan composing performance records + training-content KG + credential wallet. ALWAYS iep:modalStatus Hypothetical — the assistant suggests, the user (and possibly their manager) decide. Tracks olke: knowledge maturity progression of the planned skill areas.
lpc:flagsCapability Object property
Capability area the performance record commented on (positive or negative). Used by the assistant to cross-link review feedback to relevant training content.
lpc:groundingFragment Object property
Content-addressed PGSL atom containing the verbatim passage. Citation = quoting this atom with its IRI.
lpc:LearnerWallet Class
Top-level container linking the user's credentials, learning history, training content, and performance records. Lives in the user's own pod; portable across employers.
lpc:LearningExperience Class
User-side record derived from an xAPI Statement. Created by ingesting Statements via applications/lrs-adapter/. Cross-links to the training content engaged with and any credential earned.
lpc:learningObjective Object property
Cross-link from training content to one of its learning objectives.
lpc:LearningObjective Class
A specific objective / lesson / passage inside training content. Carries the grounding fragment (PGSL atom) the assistant cites.
lpc:PerformanceRecord Class
A review / rating / feedback record. Provenance attributes it to the issuer (manager, peer, self). Lives in the user's pod with full iep:TrustFacet.
lpc:relatesToContent Object property
Cross-link from a learning experience to the lpc:TrainingContent / lpc:LearningObjective that was engaged with.
lpc:relatesToCredential Object property
Cross-link from a learning experience to a lpc:Credential it earned (when applicable).
lpc:reviewType Object property
lpc:ReviewType Class
lpc:TrainingContent Class
Authoritative training material (course / module / lesson) ingested as a knowledge graph for grounded RAG. Source of citations the assistant returns.
lpc:vcProof Datatype property
JSON-encoded VC proof block (proofValue + proofPurpose + verificationMethod). Stored verbatim so verification is reproducible.
lpc:verificationStatus Object property
lpc:VerificationStatus Class
Outcome of cryptographic verification of a credential's proof block.
