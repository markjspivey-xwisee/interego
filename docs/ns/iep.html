Interego Protocol vocabulary iep:

This is the namespace document for https://markjspivey-xwisee.github.io/interego/ns/iep# — the Interego Protocol (L1) vocabulary: the classes and properties an agent uses to publish, compose, and reason over context on the substrate. Terms are identified by hash IRIs under this base (e.g. iep:modalStatus = …/ns/iep#modalStatus).

Machine-readable form: iep.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it.

Selected terms

iep:modalStatus — the epistemic modality of an assertion (Asserted vs Hypothetical). Used as an xAPI result.extensions key.
iep:coherentNarratives — the set of coherent source narratives a projection preserved.
iep:projectionLossy — xsd:boolean; true when a projection dropped information relative to its source.
iep:sourceDescriptor — the dereferenceable descriptor a projected artifact was derived from (chain of custody).

The complete term set — with domains, ranges, and iep:constructedFrom provenance — is in iep.ttl.
