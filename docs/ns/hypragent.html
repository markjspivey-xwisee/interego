HyprAgent — Agent Machinery for HyprCat Federation hypragent:

Namespace https://markjspivey-xwisee.github.io/interego/ns/hypragent# · 12 terms.

Agent-side ontology matching HyprCat's data-catalog side. Models cross-world delegation, affordance consumption/production, and dispatch-time policy evaluation. Aligns with ieh:Affordance + iep:AuthorizedAgent + prov:SoftwareAgent; extends them with federation-native semantics so an agent that crosses hyprcat:Worlds carries its authorization visibly.

Machine-readable form: hypragent.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. hypragent:actsOnBehalfOf = https://markjspivey-xwisee.github.io/interego/ns/hypragent#actsOnBehalfOf).

Terms

hypragent:actsOnBehalfOf Object property
The hypragent:Principal this agent ultimately represents. Chains through hypragent:DelegationChain. Equivalent to prov:actedOnBehalfOf but typed to HyprAgent semantics.
hypragent:Agent Class
A software agent participating in HyprCat federation. Every HyprAgent has (a) a stable DID identity, (b) an X25519 encryption keypair registered on at least one pod, (c) a hyprcat:world membership, (d) an optional set of hypragent:consumable / hypragent:producible affordances it is registered to handle. HyprAgents are the actors that invoke HyprCat affordances.
hypragent:Capability Class
A capability an agent is registered to handle — consuming or producing a particular class of affordance. Enables hyprcat:FederatedCatalog queries like 'list agents that can iep:canDecrypt a iep:EncryptedGraphEnvelope'. A hypragent:Agent typed with hypragent:canDecryptEnvelope can be dispatched against encrypted distributions; agents without the capability are filtered out at routing time.
hypragent:consumable Object property
A class of affordance this agent is registered to consume (e.g. hypragent:canDecryptEnvelope). Catalog clients filter for agents with matching consumables when routing requests.
hypragent:delegationChain Object property
Ordered list of Verifiable Credentials proving authority from Principal down to the current actor. Evaluated at dispatch time.
hypragent:DelegationChain Class
The sequence of W3C Verifiable Credentials that prove authority from a Principal through zero or more delegating HyprAgents down to the current actor. Consumed at dispatch time by a policy engine (ieh:PolicyDecision) to answer 'does this actor have authority to invoke this affordance?'. Every link in the chain is a iep:delegationCredential.
hypragent:Dispatch Class
A single recorded invocation of a hyprcat:FederatedDistribution by a hypragent:Agent. Carries the delegation chain that was evaluated, the policy decision result, and a provenance trail tying input ports to output ports. Gives federation auditability end-to-end.
hypragent:dispatchedOn Object property
The hyprcat:FederatedDistribution this dispatch activity invoked. Domain/range ties dispatch records to the specific catalog entry they acted against.
hypragent:policyDecision Object property
The ieh:PolicyDecision that authorized (or blocked) this dispatch. Gives an audit trail from 'what was attempted' -> 'which policy evaluated' -> 'what rule matched'.
hypragent:Principal Class
The human (or organizational) entity an agent ultimately acts on behalf of. A delegation chain roots in a Principal and flows through zero or more intermediate HyprAgents to the acting agent. For Interego, the Principal is typically the pod owner's WebID.
hypragent:producible Object property
A class of affordance this agent is registered to produce (e.g. hypragent:canPublishContext). Declares write-side capabilities for catalog routing.
hypragent:WorldCrossing Class
A hypragent:Dispatch where the actor's hyprcat:world differs from the target resource's hyprcat:world. These get extra scrutiny from policy engines because they represent trust-boundary traversals (e.g. an AgentWorld bot reading a UserWorld pod resource). Logged + policy-evaluated; often require explicit cross-world delegation credentials.
