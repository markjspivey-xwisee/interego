Namespace https://markjspivey-xwisee.github.io/interego/ns/nist-rmf# · 19 terms.

L3 mapping of Interego context primitives to NIST AI RMF (AI 100-1) four functions: Govern / Map / Measure / Manage.

Machine-readable form: nist-rmf.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. nist-rmf:contributesTo = https://markjspivey-xwisee.github.io/interego/ns/nist-rmf#contributesTo).

Terms

nist-rmf:contributesTo Object property
Links an evidence descriptor to one or more category IRIs. Many-to-many: one record can satisfy multiple categories, one category needs multiple records.
nist-rmf:evidenceLevel Datatype property
How strong this evidence is: 'asserted' / 'measured' / 'audited' / 'externally-attested'.
nist-rmf:EvidenceRecord Class
A descriptor that contributes evidence toward one or more RMF function categories. Carries nist-rmf:contributesTo links to the specific category IRIs (e.g., nist-rmf:Map.1.2). Compliance reports aggregate evidence per category.
nist-rmf:functionContext Object property
The function (Govern/Map/Measure/Manage) under which this evidence applies.
