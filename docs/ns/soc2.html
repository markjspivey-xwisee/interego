SOC 2 Trust Services Criteria mapping ontology soc2:

Namespace https://markjspivey-xwisee.github.io/interego/ns/soc2# · 60 terms.

L3 mapping of Interego context primitives to AICPA SOC 2 TSC. Covers five categories with selected controls; extensible.

Machine-readable form: soc2.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. soc2:accessAction = https://markjspivey-xwisee.github.io/interego/ns/soc2#accessAction).
