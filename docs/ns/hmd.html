HyperMarkdown Vocabulary hmd:

Namespace https://relay.interego.xwisee.com/ns/maintainer/hmd# · version 0.2-draft · 36 terms.

Terms for HyperMarkdown 0.1 — Markdown as a hypermedia media type: documents as graph resources, typed links as edges, semantic claims, validation shapes, provenance, policy, and controls as discoverable affordances. A domain vocabulary that ALIGNS with the Interego protocol layer (iep:) and W3C hypermedia/linked-data vocabularies via rdfs:seeAlso and skos:related; it does not subclass or extend them.

Machine-readable form: hmd.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. hmd:ApprovalRequest = https://relay.interego.xwisee.com/ns/maintainer/hmd#ApprovalRequest).

Terms

hmd:ApprovalRequest Class
Input payload type expected by an approve control.
hmd:approve Property
Approve the target resource, advancing its state.
hmd:ApprovedResource Class
Output payload type returned by an approve control.
hmd:cell Object property
Relates a row to one of its cells. Column order is carried by hmd:colIndex, not by statement order.
hmd:Cell Class
One cell of an hmd:Row: its text (hmd:cellText) at a 0-based hmd:colIndex. The text is the cell's CommonMark inline source verbatim; any inline markup or typed link inside it is rendered (and XSS-hardened) at view time, never lifted as a control edge.
hmd:CellShape Shape
hmd:cellText Datatype property
The text content of a cell — the CommonMark inline source of the pipe-table cell, verbatim.
hmd:claim Object property
Relates a document to a semantic claim it asserts.
hmd:Claim Class
An asserted statement (subject/predicate/object) that may carry confidence, source, and provenance; reifiable as an rdf:Statement.
hmd:colIndex Datatype property
The 0-based ordinal position of a cell within its row.
hmd:condition Property
A state precondition under which a control is available.
hmd:control Object property
Relates a document (or resource) to a control it exposes.
hmd:Control Class
A hypermedia control: a declared, discoverable action or state transition available on a resource. Clients must not assume an action is available unless a corresponding control is present.
hmd:Document Class
A HyperMarkdown document viewed as a locally-editable representation of a distributed graph resource: prose, typed links, semantic blocks, and controls.
hmd:expects Object property
The expected input type of a control.
hmd:header Datatype property
True when a row is the table's header row. Absent (or false) on data rows.
hmd:method Datatype property
The HTTP (or local) method a control is invoked with.
hmd:reject Property
Reject the target resource, advancing its state.
hmd:RejectedResource Class
Output payload type returned by a reject control.
hmd:RejectionNotice Class
Input payload type expected by a reject control.
hmd:rel Property
The semantic action relationship a control realises.
hmd:reportSymptom Property
Submit a symptom report against the target resource.
hmd:returns Object property
The expected output type of a control.
hmd:row Object property
Relates a table to one of its rows. Row order is carried by hmd:rowIndex, not by statement order.
hmd:Row Class
One row of an hmd:Table: an ordered set of hmd:Cell at a 0-based hmd:rowIndex. The header row carries hmd:header true; data rows omit it (or set it false).
hmd:rowIndex Datatype property
The 0-based ordinal position of a row within its table (0 is the header row).
hmd:RowShape Shape
hmd:search Property
A parameterised query relation for templated links.
hmd:SymptomReport Class
Input payload type for a report-symptom control.
hmd:SymptomTimeline Class
Output payload type returned by a report-symptom control.
hmd:table Object property
Relates a document to a table lifted from a pipe table in its body.
hmd:Table Class
Tabular data lifted from a GFM pipe table in a document's CommonMark body: an ordered set of hmd:Row, the first flagged as the header. DATA, not a control — carries no authority and no transport endpoint.
hmd:TableShape Shape
hmd:target Object property
The URI a control is invoked against.
hmd:template Object property
Relates a document to a parameterised-link template it exposes.
hmd:Template Class
A parameterised link: an RFC 6570 URI template with variable-to-property mappings.
