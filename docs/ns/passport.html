Namespace https://markjspivey-xwisee.github.io/interego/ns/passport# · 15 terms.

Persistent agent biographical identity that survives infrastructure changes. Built on iep:ContextDescriptor + amta:Attestation + registry:RegistryEntry.

Machine-readable form: passport.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. passport:agentIdentity = https://markjspivey-xwisee.github.io/interego/ns/passport#agentIdentity).

Terms

passport:agentIdentity Object property
The agent's stable cryptographic identity (DID/WebID). Same identity across all infrastructure migrations.
passport:CapabilityAcquisition Class
A LifeEvent recording when the agent first demonstrated a capability — the action descriptor that proves it, plus any attestations that corroborated.
passport:DelegationCredential Class
A credential, issued by a human owner (or a delegating agent), that authorizes an agent to act on the owner's behalf within a bounded scope (iep:scope). Every cross-agent action references the DelegationCredential that authorized it; receivers validate the action falls within scope and reject outside-scope actions. Carried in the passport so an agent's standing authority travels with its biography. Built on iep:AuthorizedAgent delegation + a iep:TrustFacet proof.

passport:demonstrationEvidence Object property
The action descriptor(s) proving the capability.
passport:hasLifeEvent Object property
passport:InfrastructureMigration Class
A specific kind of LifeEvent: the agent moved from one framework / pod / model to another, with continuity of identity. Records the from/to + the cryptographic proof that the same DID owns both.
passport:LifeEvent Class
A milestone in the agent's biography — registration on a registry, earning an attestation, completing a notable action, switching infrastructure. Each LifeEvent cites its evidence (descriptor URLs) so the passport is auditable end-to-end.
passport:livesAt Object property
Current home pod URL where the canonical passport version is published.
passport:newInfrastructure Datatype property
Identifier for the framework/runtime/pod the agent migrated TO.
passport:Passport Class
An agent's biographical identity. A specialized iep:ContextDescriptor whose graph contains the agent's lifespan, accumulated attestations, completed actions, registered capabilities, and stated values. Versioned via iep:supersedes; signed by the agent's stable DID/WebID.
passport:previousIdentity Object property
Identity used before this migration, if different from the current one (e.g., legacy username before DID adoption).
passport:registeredOn Object property
Public registries this agent is listed on. Lets a passport-reader find aggregated reputation across the agent's active registries.
passport:statedValue Object property
passport:StatedValue Class
A value the agent commits to (e.g. "prefer cited sources", "refuse off-topic requests", "escalate uncertain claims to human review"). Carried in the passport as Asserted-modal SemioticFacet so violators are detectable post-hoc by comparing actions to stated values.
