AMTA — Agent-Mediated Trust Attestation amta:

Namespace https://markjspivey-xwisee.github.io/interego/ns/amta# · 18 terms.

Multi-dimensional trust attestations: competence, honesty, relevance, recency. Each axis is rated ordinally; combined attestation is a vector usable in policy decisions. Stored as iep:TrustFacet specializations.

Machine-readable form: amta.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. amta:Attestation = https://markjspivey-xwisee.github.io/interego/ns/amta#Attestation).

Terms

amta:Attestation Class
A multi-dimensional trust attestation by one agent (the attestor) about another agent or claim (the subject). Subclass of iep:TrustFacet; inherits trust-level + issuer semantics and adds per-axis ratings.
amta:attestedAt Datatype property
amta:attestor Object property
amta:attestsTo Object property
The capability the attestation concerns — typically a tool, teaching package, or other artifact IRI. The attestation rates the subject's competence/honesty/etc. with respect to this capability.
amta:axis Object property
The trust axis this attestation rates (e.g. correctness, efficiency, safety, generality). Lets a single attestation declare which dimension its amta:rating applies to, rather than carrying every per-axis property.
amta:clearance Datatype property
A subject's clearance level on a monotone classification lattice (higher dominates). Read by the ABAC policy-decision point as a SHACL minInclusive constraint: a request is permitted only when the subject's amta:clearance meets or exceeds the resource's classification.
amta:competence Datatype property
amta:CompetenceRating Class
Rating of the subject's ability to produce correct claims in the relevant domain. Ordinal 0.0 - 1.0. A lens on iep:SemioticFacet.epistemicConfidence filtered to the attestor's domain.
amta:fromExecution Object property
The execution-evidence descriptor that grounds this attestation — the recorded tool execution / verified transfer the rating was derived from. Makes the attestation auditable rather than self-asserted.
amta:honesty Datatype property
amta:HonestyRating Class
Rating of the subject's faithful reporting. Specialization of iep:TrustFacet for a per-axis honesty signal.
amta:rating Datatype property
A single normalized rating value (0.0 - 1.0) for the axis named by amta:axis. Used when an attestation rates one axis at a time rather than carrying the full per-axis vector.
amta:recency Datatype property
amta:RecencyRating Class
Rating derived from time-since-attestation. Constructed from iep:TemporalFacet.validFrom + an attestation's provenance timestamp.
amta:relevance Datatype property
amta:RelevanceRating Class
Rating of whether the subject's expertise is germane to the specific claim being attested.
amta:Reputation Class
The aggregate of attestations an agent has received. Constructed at runtime from a set of amta:Attestation descriptors via statistical aggregation; see examples/reputation-aggregator.mjs for a reference realization.
