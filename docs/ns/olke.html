OLKE — Organizational Learning & Knowledge Evolution olke:

Namespace https://markjspivey-xwisee.github.io/interego/ns/olke# · 7 terms.

Stages and transitions by which knowledge evolves within an organization: tacit -> articulate -> collective -> institutional. Annotates iep:ContextDescriptor with the current stage + allows tracking of stage transitions.

olke:KnowledgeStage Class
Abstract parent of the four canonical OLKE stages. A stage is a classification on iep:ContextDescriptor (carried via olke:stage); knowledge stages are epistemic lenses over descriptors.
