Vault-LD conformance profile vault-ld:

Namespace https://relay.interego.xwisee.com/ns/maintainer/vault-ld# · version 0.1-draft · 7 terms.

A declarative projection profile that makes the general Markdown-vault engine conform to Vault-LD v0.5. Its knobs (below) select the rung ceiling, context-resolution, identity minting, link grammar, and placement predicate; its SHACL shape (vldp:ConformanceShape) verifies the rung-<=3 authority closure on the projected graph.

Machine-readable form: vault-ld.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. vault-ld:conformanceShape = https://relay.interego.xwisee.com/ns/maintainer/vault-ld#conformanceShape).

Terms

vault-ld:conformanceShape Object property
The SHACL NodeShape a projection's output must satisfy to conform to this profile.
vault-ld:ConformanceShape Shape
Validates the rung-<=3 authority closure over a projected Vault-LD graph: no executable operation/affordance/control types, and no authority predicates.
vault-ld:EntailmentAuthorityShape Shape
Refuses any node whose rdf:type reaches an authority class, or which uses a predicate that reaches an authority predicate, through rdfs:subClassOf / owl:equivalentClass / rdfs:subPropertyOf / owl:equivalentProperty / owl:sameAs chains.
vault-ld:maxRung Datatype property
The HyperMarkdown progressive-enhancement ceiling the profile permits. 3 = descriptive data only (no executable affordances); 4 = full HMD. Enforced by vldp:ConformanceShape.
vault-ld:NoAuthorityPredicatesShape Shape
No subject may carry an authority predicate (an operation target/method, a supported operation, an action, or an HMD control edge).
vault-ld:NoExternalImportShape Shape
A rung-<=3 vault must be self-contained: owl:imports pulls in an external TBox the authority floor cannot screen, which could name authority for a downstream import-reasoner. Vault-LD composes local context.jsonld (@context), not OWL imports.
vault-ld:Profile Class
A named set of knobs (rung ceiling, context source, identity strategy, link grammar, placement predicate) + a conformance shape that make the general Markdown-vault projection engine behave as a specific dialect. Data, not code.
