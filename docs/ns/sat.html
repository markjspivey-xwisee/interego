SAT — Semiotic Agent Topos sat:

Namespace https://markjspivey-xwisee.github.io/interego/ns/sat# · 16 terms.

Topos-theoretic model of agents as participants in a semiotic field. The Semiotic Field Functor Σ maps situations to possibility-spaces of meaning; every iep:SemioticFacet is a projection of Σ onto a specific context graph. Complements Peircean triadic semiotics with sheaf structure and endofunctor semiosis.

sat:dampedBy Object property
Identifies an intervention type that, if applied, would suppress the disposition's actualization without claiming to address its underlying conditions. The honest framing: damping buys time, not resolution.
sat:Disposition Class
A typed tendency — what a system is currently disposed to do under its present constraints. Distinct from a prediction: a disposition is a structured affordance for next-state action that may or may not actualize. Subject to sat:weakSignal (what surfaces it early), sat:shiftedBy (what alters it), and sat:dampedBy (what suppresses it).
sat:dispositionalConfidence Datatype property
Confidence in [0.0, 1.0] that the system is currently disposed as claimed. NOT a probability of actualization; a confidence in the disposition's existence. Pair with iep:modalStatus = Hypothetical for honest semiotics: dispositions are assertions about latent tendency, not about future state.
sat:EmergentMeaning Class
Meaning arising from multi-agent semiosis; sheafification of interpretant assignments. Constructed from a set of sat:Interpretant over a sat:SemioticField via categorical colimit.
sat:hasDisposition Object property
Attaches a typed disposition to the situation a descriptor describes. A descriptor may carry multiple sat:hasDisposition values — they are not mutually exclusive (a system can be disposed to multiple courses simultaneously, with different probabilities or under different sub-conditions).
sat:Interpretant Class
Peircean interpretant — the agent's instantiated meaning-assignment. Every iep:SemioticFacet IS an interpretant (the facet carries the modal status + confidence that realize the interpretant for a specific sign).
sat:Object Class
The thing the sign is about — external referent. Primitive (external to the category of iep: descriptors); in a context graph this is typically the named-graph IRI (iep:describes target).
sat:possibleInterpretants Object property
sat:Semiosis Class
The process of interpretation itself — endofunctor iterating triadic sign relations until a stable interpretant emerges. Constructed by iterating sat:SemioticFieldFunctor over sat:Interpretant.
sat:SemioticField Class
The output of Σ(situation) — the space of meaning-making possibilities available in a given context.
sat:SemioticFieldFunctor Class
The functor Σ : Situations → SemioticFields that defines the semiotic topos. Projection of Σ onto a specific graph yields a iep:SemioticFacet.
sat:shiftedBy Object property
Identifies an intervention type that, if applied, would alter the disposition. Distinct from a 'fix': the disposition isn't broken; it can be redirected. Used for amplify/dampen routing in Cynefin-Complex domains.
sat:Sign Class
Peircean representamen — the signal that gets interpreted. Every iep:ContextDescriptor IS a sign (graph = representamen, object = referent, facets = interpretant).
sat:Situation Class
A context in which meaning-making occurs. Inputs to Σ; each situation has a Semiotic Field of possible interpretations, actions, and meaning assignments.
sat:weakSignal Object property
Links a disposition to a descriptor that, if observed, would surface the disposition early — before it actualizes. Snowden's 'dispositional shift' detection mechanism. The signal need not be predictive on its own — its semiotic work is to make a latent tendency legible.
