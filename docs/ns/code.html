code: — Source code domain ontology code:

Namespace https://markjspivey-xwisee.github.io/interego/ns/code# · 25 terms.

L3 domain ontology for source-code artifacts. Every class is grounded in L1 (iep:/pgsl:) or a W3C vocabulary. Demonstrates that a non-trivial software domain can be expressed compositionally on top of the Interego protocol without new primitives.

Machine-readable form: code.ttl (Turtle) — the authoritative source. This page is a human-readable projection of it; each term below is identified by a hash IRI under this base (e.g. code:authoredBy = https://markjspivey-xwisee.github.io/interego/ns/code#authoredBy).

Terms

code:authoredBy Object property
The agent who authored a commit. Specialises prov:wasAttributedTo for the code domain.
code:Branch Class
A named line of commits. A branch IS a paradigmatic selection over the space of possible commit sequences for a repository — the other branches are the paradigm alternatives.
code:branchName Datatype property
code:buildOutcome Datatype property
The terminal state of a build: success | failure | cancelled.
code:BuildResult Class
Outcome of a build (pass/fail, artifact produced, logs). An attestation: who vouches for this outcome (trust facet) + when the build activity ran (provenance facet).
code:Commit Class
A single commit — one granular unit in a commit sequence. Modeled as pgsl:Fragment (granular unit in a sequence lattice) since commit chains are naturally syntagmatic; parent/child relations are pgsl:pullback-compatible.
code:commitMessage Datatype property
code:commitSha Datatype property
code:Defect Class
A reported bug or flaw. Constructed from a semiotic facet whose interpretant is 'this artifact is buggy' with severity + epistemic confidence. Modal state tracks whether the defect is Asserted (confirmed), Hypothetical (suspected), or Counterfactual (refuted).
code:linesAdded Datatype property
code:linesRemoved Datatype property
code:onRepository Object property
The repository a PR / commit / branch belongs to.
code:parentCommit Object property
The immediate ancestor of a commit in the syntagmatic chain. A commit may have multiple parents (merge commits).
code:PullRequest Class
A change proposal. Extends iep:ContextDescriptor directly because a PR is inherently a typed context that carries temporal (opened/merged), provenance (author), agent (reviewers), semiotic (open/closed/merged modal), trust (required approvals), and federation (which repo + pod) facets.
code:reportedIn Object property
The commit or pull request in which a defect was identified.
code:Repository Class
A source-code repository. Extends dcat:Dataset with the protocol's federation model (each repo surfaces as a typed, federated dataset with commits as its primary records).
code:Review Class
A review of a pull request. Constructed from a semiotic facet (the verdict + modal state + confidence) and a provenance facet (who reviewed, when). Composes with other reviews via the meet (most-conservative) on the modal lattice.
code:reviewedBy Object property
Associates a pull request with a code review. Multiple reviews per PR compose via the modal lattice.
code:ReviewVerdict Class
The modal-typed conclusion of a code review: Approved (Asserted the PR is mergeable), ChangesRequested (Counterfactual — in current form NOT mergeable), or Commented (Hypothetical — observations without gate).
code:severity Object property
The severity level of a reported defect.
code:Severity Class
A defect severity ranking (Critical > High > Medium > Low). Primitive enumeration — the values themselves are the unit of meaning, no further decomposition needed at this layer.
code:targetsBranch Object property
The branch a pull request proposes to merge into.
code:testedArtifact Object property
The commit or PR that a test run executed against.
code:TestRun Class
An execution of a test suite against a commit or PR. A prov:Activity whose outputs include build results + test outcomes. Provenance facet on descriptors carries start/end times + invoking agent.
code:verdict Object property
The modal-typed conclusion of a review.
